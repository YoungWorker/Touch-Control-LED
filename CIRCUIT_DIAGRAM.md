## 接线说明

### TTP223 触摸传感器 接线
```
TTP223 模块
  ├─ VCC ──→ ESP8266 5V (或 3.3V)
  ├─ GND ──→ ESP8266 GND
  └─ OUT ──→ ESP8266 D5 (GPIO14)
```

### LED 接线（方案 A：直接驱动 - 适合低功率LED）
```
冷白色 LED
  ├─ 正极 ──→ 220Ω 电阻 ──→ ESP8266 D2 (GPIO4)
  └─ 负极 ──→ ESP8266 GND

暖色光 LED
  ├─ 正极 ──→ 220Ω 电阻 ──→ ESP8266 D1 (GPIO5)
  └─ 负极 ──→ ESP8266 GND

暖白混合 LED
  ├─ 正极 ──→ 220Ω 电阻 ──→ ESP8266 D3 (GPIO0)
  └─ 负极 ──→ ESP8266 GND
```

### LED 接线（方案 B：三极管驱动 - 适合高功率LED）

**以冷白色 LED 为例**：
```
冷白色 LED
  ├─ 正极 ──→ 220Ω 电阻 ──→ 3.3V
  └─ 负极 ──→ 2N7000 三极管集电极

2N7000 三极管
  ├─ 基极 ──→ 220Ω 电阻 ──→ ESP8266 D2 (GPIO4)
  ├─ 发射极 ──→ ESP8266 GND
  └─ 集电极 ──→ LED 负极
```

## ESP8266 D1 Mini 引脚映射

```
┌─────────────────────────────────────┐
│     ESP8266 D1 Mini 开发板          │
├─────────────────────────────────────┤
│ RST    GND    D0    D1              │  D1 (GPIO5)   → 暖白色 LED
│ A0     D5     D6    D7              │  D2 (GPIO4)   → 冷白色 LED
│ D8     D4     D3    D2              │  D3 (GPIO0)   → 暖白混合 LED
│ 3.3V   GND    TX    RX              │  D5 (GPIO14)  → TTP223 触摸传感器
└─────────────────────────────────────┘

引脚详细说明：
D0 (GPIO16) - 不可用（用于复位）
D1 (GPIO5)  - ✓ 暖白色 LED PWM 输出
D2 (GPIO4)  - ✓ 冷白色 LED PWM 输出
D3 (GPIO0)  - ✓ 暖白混合 LED PWM 输出
D4 (GPIO2)  - LED_BUILTIN（板载LED）
D5 (GPIO14) - ✓ TTP223 触摸输入
D6 (GPIO12) - 可用
D7 (GPIO13) - 可用
D8 (GPIO15) - 不可用（启动时必须为LOW）

RX (GPIO3)  - 串口接收
TX (GPIO1)  - 串口发送
3.3V        - 电源 3.3V
GND         - 地
A0 (ADC0)   - 模拟输入
```

## 功率计算

### 单个 LED 功率
```
假设每个 LED 规格：5mm 20mA @ 3V

LED 功率 = I × V = 0.02A × 3V = 0.06W
限流电阻功率 = 0.02A × 0.02A × 220Ω ≈ 0.088W (1/8W)
```

### 总系统功率
```
三个 LED 最大电流：3 × 20mA = 60mA
ESP8266 工作电流：~80mA
总最大电流：~140mA

推荐电源规格：
- USB 电源：5V 1A 以上
- DC 适配器：5V 1.5A 以上
```

## 调试步骤

### 1. 硬件检查
```
□ 检查所有接线是否牢固
□ 确认 ESP8266 能正常供电（LED_BUILTIN 闪烁）
□ 测量各引脚电压是否正常
□ 检查 LED 极性是否正确
```

### 2. 传感器测试
```
□ 用万用表测量 TTP223 OUT 脚电压
  - 未触摸时：应为 LOW (~0V)
  - 触摸时：应跳变到 HIGH (~3.3V)
□ 检查防抖时间设置（推荐 300ms）
```

### 3. LED 测试
```
□ 逐个测试每个 LED
□ 检查 PWM 输出波形（示波器）
□ 验证亮度变化是否正常
□ 检查色温混合效果
```

### 4. 程序验证
```
□ 打开串口监视器（波特率 115200）
□ 触摸传感器，检查是否有输出
□ 验证触摸计数是否正确
□ 检查 LED 是否按照期望改变
```

## 常见接线错误

### ❌ 错误 1：LED 极性反接
```
症状：LED 不亮
解决：确认 LED 正极（长脚）接向电阻
```

### ❌ 错误 2：限流电阻遗漏
```
症状：LED 烧坏，GPIO 烧毁
解决：必须在 LED 和 GPIO 之间串联 220Ω 电阻
```

### ❌ 错误 3：传感器接线错误
```
症状：触摸无反应
解决：
- 检查 OUT 脚是否接到 D5 (GPIO14)
- 确认 VCC/GND 接线正确
- TTP223 有两种接法，检查是否为触摸模式
```

### ❌ 错误 4：浮空引脚
```
症状：不稳定/随机状态
解决：确保所有未使用的引脚悬空或接 GND
```

## 电源连接建议

### 方案 A：USB 直连（推荐新手）
```
USB 5V 适配器 → ESP8266 microUSB 接口
优点：简单方便
缺点：需要 USB 线长时间连接
```

### 方案 B：USB 转 5V 引脚
```
USB 5V ──→ ESP8266 5V 引脚
USB GND ──→ ESP8266 GND 引脚
优点：可以断线使用
缺点：需要焊接
```

### 方案 C：外接 DC 电源
```
DC 5V 适配器 ──→ ESP8266 VIN 或 5V 引脚
DC GND ──→ ESP8266 GND
优点：可以长期使用
缺点：需要额外的电源适配器
```

## 性能优化建议

### 1. 降低功耗
```
- 使用更小的 LED（3mm）
- 降低 PWM 频率到 500Hz
- 实现亮度自动调节
```

### 2. 提高亮度
```
- 使用三极管驱动（参考方案 B）
- 增加 LED 供电电压到 5V
- 使用更高功率的 LED（1W）
```

### 3. 改善色温
```
- 使用 RGB LED 代替多个单色 LED
- 调整 PWM 比例实现更准确的色温
- 添加色温传感器反馈
```

## 安全提示

⚠️ **重要事项**

1. **避免短路**
   - 连接前检查接线
   - 确保 VCC 和 GND 不会直接接触

2. **防止烧坏 GPIO**
   - 必须使用限流电阻
   - 不要直接驱动大电流

3. **静电保护**
   - 接线前接触接地**
   - 使用 ESD 腕带

4. **电源管理**
   - 使用稳压电源
   - 添加滤波电容防止干扰
