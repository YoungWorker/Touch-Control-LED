# Touch-Control-LED

一种基于 ESP8266 的触控智能灯，具备可调色温功能。

## 项目特性

✨ **核心功能**
- 🎯 **触摸计数控制**：通过不同次数的触摸实现不同功能
- 💡 **三色可切换**：白光、暖色光、冷白色光三种模式
- 🔌 **低功耗设计**：基于 ESP8266 微控制器
- 📊 **串口调试**：实时显示状态信息
- ⏱️ **智能超时**：2秒无触摸自动重置

## 工作原理

### 触摸控制逻辑

| 触摸次数 | 功能 | LED 状态 |
|--------|------|--------|
| 第1次 | 打开白光 | 暖白色混合 |
| 第2次 | 切换暖色光 | 仅暖色 LED |
| 第3次 | 切换冷白色光 | 仅冷白 LED |
| 第4次 | 关灯 | 全灭 |
| 2秒无触摸 | 重置计数 | 保持当前状态 |

### 状态机

```
初始(灯灭) → [触摸1] → 白光
         ↓
白光      → [触摸2] → 暖色光
         ↓
暖色光    → [触摸3] → 冷白色光
         ↓
冷白色光  → [触摸4] → 灯灭
         ↓
灯灭      → [2秒超时] → 重置计数
```

## 硬件需求

### 必需元件

| 元件 | 型号 | 数量 | 说明 |
|------|------|------|------|
| 微控制器 | ESP8266 D1 Mini | 1 | 主控制器 |
| 触摸传感器 | TTP223 模块 | 1 | 电容式触摸检测 |
| LED（冷白） | 5mm 白色 LED | 1 | 冷白色光 |
| LED（暖白） | 5mm 暖白 LED | 1 | 暖色光 |
| LED（混合） | 5mm 暖白 LED | 1 | 暖白混合 |
| 限流电阻 | 220Ω 1/4W | 3 | LED 限流 |
| 电源 | USB 5V 1A | 1 | 系统电源 |

### 可选元件

- 三极管 2N7000 × 3（用于高功率 LED 驱动）
- 电容 10μF/25V（电源滤波）

## 电路连接

### 引脚定义

```
ESP8266 D1 Mini
┌─────────────────────┐
│ RST   GND   D0  D1  │
│ A0    D5    D6  D7  │
│ D8    D4    D3  D2  │
│ 3V3   GND   TX  RX  │
└─────────────────────┘

D1 (GPIO5)   → 暖白色 LED (+ 220Ω)
D2 (GPIO4)   → 冷白色 LED (+ 220Ω)
D3 (GPIO0)   → 混合 LED (+ 220Ω)
D5 (GPIO14)  → TTP223 传感器 OUT
GND          → 公共地
3.3V/5V      → 电源
```

### 接线图

**TTP223 触摸传感器**
```
TTP223 模块
  ├─ VCC → ESP8266 5V (或 3.3V)
  ├─ GND → ESP8266 GND
  └─ OUT → ESP8266 D5 (GPIO14)
```

**LED 接线（直接驱动版）**
```
冷白 LED
  ├─ 正极 → 220Ω 电阻 → ESP8266 D2 (GPIO4)
  └─ 负极 → GND

暖白 LED（第1个）
  ├─ 正极 → 220Ω 电阻 → ESP8266 D1 (GPIO5)
  └─ 负极 → GND

暖白 LED（第2个）
  ├─ 正极 → 220Ω 电阻 → ESP8266 D3 (GPIO0)
  └─ 负极 → GND
```

## 快速开始

### 1. 环境搭建

#### 安装 Arduino IDE
- 下载：https://www.arduino.cc/en/software
- 版本：1.8 或更高

#### 添加 ESP8266 开发板支持
1. 打开 Arduino IDE
2. 文件 → 首选项
3. 附加开发板管理器网址，添加：
   ```
   http://arduino.esp8266.com/stable/package_esp8266com_index.json
   ```
4. 工具 → 开发板 → 开发板管理器
5. 搜索 "ESP8266"，安装最新版本

#### 配置开发板
1. 工具 → 开发板 → 选择 "LOLIN(WEMOS) D1 Mini"
2. 工具 → 上传速度 → 115200
3. 工具 → 端口 → 选择 COM 端口

### 2. 上传代码

1. 打开 `Touch_Control_LED.ino`
2. 点击 ⬆️ 上传按钮
3. 等待编译和上传完成

### 3. 调试

1. 打开 Arduino IDE 串口监视器
2. 波特率设置为 **115200**
3. 查看实时输出信息

```

  第1次触摸 → 白光 (暖白色光模式)
  第2次触摸 → 暖色光
  第3次触摸 → 冷白色光
  第4次触摸 → 关灯
  2秒无触摸则重新计数

Setup complete. Waiting for touch input...

[触摸 #1] 检测到触摸

  →白光打开 (暖白色光)
```

## 使用说明

### 基本��作

1. **打开白光**
   - 触摸传感器第 1 次
   - 灯亮起，显示白光色温

2. **切换暖色光**
   - 再触摸 1 次（第 2 次总计）
   - 灯变为暖色光

3. **切换冷白色光**
   - 再触摸 1 次（第 3 次总计）
   - 灯变为冷白色光

4. **关闭灯光**
   - 再触摸 1 次（第 4 次总计）
   - 灯关闭，计数重置

5. **重新开始**
   - 2 秒内不触摸，计数自动重置
   - 下次触摸重新从第 1 次开始

### 工作示例

```
场景：用户想看白光、暖色、冷白、关灯

时间      触摸      灯光状态        说明
─────────────────────────────────────
0ms       触摸      ✓ 白光          第1次触摸
500ms     触摸      ✓ 暖色光        第2次触摸
1000ms    触摸      ✓ 冷白色光      第3次触摸
1500ms    触摸      ✗ 灯灭         第4次触摸
3500ms    -         ✗ 灯灭         2秒超时，计数重置
4000ms    触摸      ✓ 白光          新序列开始，第1次
```

## 常见问题

### Q: 触摸传感器没反应
**A:**
- 检查 TTP223 供电是否正常
- 确认 OUT 脚正确连接到 D5
- 用万用表测量 OUT 脚电压变化
- 尝试靠近触摸区域

### Q: LED 不亮
**A:**
- 检查 LED 极性（正极接向电阻）
- 确认 GPIO 输出是否正常
- 检查限流电阻是否正确（220Ω）
- 测试 LED 本身是否完好

### Q: 色温不准确
**A:**
- 调整 PWM 输出值（在 `updateLED()` 中）
- 检查 LED 质量和色温规格
- 考虑使用三极管驱动以提高功率

### Q: 计数器不重置
**A:**
- 检查 `TOUCH_TIMEOUT` 是否设置正确（2000ms）
- 查看串口输出确认是否收到信号
- 验证系统时间是否正常
